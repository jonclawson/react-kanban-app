# Kanban app
## React + TypeScript + Vite

This is an example dragable kanban app built with React. Allows user to create tasks with title, description and "Rick and Morty Character".

- fetch call to the Rick and Morty GraphQL API (https://rickandmortyapi.com/graphql) to populate a list of characters
- Items are created via a form - each item must have a Rick and Morty character assigned to it
- Items are draggable between columns and reorderable within a column
- When an item is moved to Done, show something delightful (confetti, animation - your call)

### Packages
- @hello-pangea/dnd
- tailwindcss
- canvas-confetti

### API used
- https://rickandmortyapi.com/graphql
